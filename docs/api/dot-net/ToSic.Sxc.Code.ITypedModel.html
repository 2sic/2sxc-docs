    Interface ITypedModel

    Object in partial Razor files to access parameters handed in. Example caller:

    @Html.Partial(someFile, new { blogPost, file = mainFile, title = &quot;hello&quot; }

    Example partial:

    var blogPost = TypedModel.Item(&quot;BlogPost&quot;);
var file = TypedModel.File(&quot;File&quot;);
var title = TypedModel.String(&quot;Title&quot;);
