Docs for Existing App Extensions

App Extensions allow you to create reusable packages which can include code, data, queries, and more, which can then be installed into multiple Apps.

This section in the documentation is dedicated to explaining how to use App Extensions which were published by 2sxc or by third-party developers.

List of App Extensions

Here is a list of some of the most popular App Extensions available for 2sxc:

  • Color Picker Spectrum: A powerful color picker input type for selecting colors with a spectrum interface.
  • Number Slider Basic: A simple number slider input type for selecting numeric values within a defined range.
  • DataSource Open Meteo: A DataSource extension for fetching weather data from the Open Meteo API.
  • Google Recaptcha V3: A code extension for integrating Google reCAPTCHA V3 into your forms for spam protection.

JavaScript Specific Extensions:

  • JS App Editions: An extension that allows you to create different editions of your JS apps for development and production.
  • Angular Loader: An extension to facilitate loading Angular applications within 2sxc.
  • React Loader: An extension to facilitate loading React applications within 2sxc.
