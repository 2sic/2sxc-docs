    Interface IConvertToJsonBasic

    Service to Convert Entities, Streams and DataSources to JsonBasic JsonEntity for automatic serialization in WebApis.

    It can convert single items like IEntity and IEntityWrapper like DynamicEntities.

    In Custom Code / Razor / WebApi you can get this service with var converter = GetService&lt;IConvertToJsonBasic>()
