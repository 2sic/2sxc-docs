    Class LookUpForTokenTemplate

    LookUp for creating token based templates. In addition to retrieving values, it also resolves special tokens like

    • repeater:index
    • repeater:isfirst
    • etc.
    Inheritance
    System.Object
    LookUpForTokenTemplate
    Implements
    ILookUp
    Namespace: ToSic.Sxc.LookUp
    Assembly: ToSic.Sxc.dll
    Syntax
    [InternalApi_DoNotUse_MayChangeWithoutNotice("this is just fyi")]
public class LookUpForTokenTemplate : ILookUp
    Remarks

    Only use this for Token templates, do not use for normal lookups which end up in data-sources. The reason is that this tries to respect culture formatting, which will cause trouble (numbers with comma etc.) when trying to use in other systems.
