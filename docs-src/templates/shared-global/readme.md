# Shared-Global Templates

This contains shared js/css etc. for use in docfx templates within the 2sxc standard.

It is generally used in these projects

1. 2sxc-docs
1. Oqtane docs
1. cre8magic docs

The master copy is maintained in the 2sxc-docs project.

## History

### Topography improvements 2026-03-07

1. Typography - improved spacing on top of headings, so they don't stick together. but skipped h1, as that is used just 1x on top and should not add extra space there.

### Docs-Tiles 2026-03-04

1. Added a new "docs-tiles" component, which is used to create the nice tiles on the "Create Your Own Extension" page. This component is now available for use in all docfx templates.

### Initial Release ca. 2023

Built everything and also documented it here: <https://docs.2sxc.org/abyss/contribute/docs/implementation/ts-css-webpack.html>
