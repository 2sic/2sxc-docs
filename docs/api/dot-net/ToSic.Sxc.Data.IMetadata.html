    Interface IMetadata

    Metadata on Dynamic Objects - like IDynamicEntity or IAsset (files/folders).

    Behaves like a normal DynamicEntity, but has additional commands to detect if specific Metadata exists.

    • Added in v13
    • Made compatible to ITypedItem in 16.02 to allow typed commands such as .String(...)
    • Renamed in v16.02 from IDynamicMetadata to IMetadata since it's not necessarily dynamic any more (but still supports dynamic where needed) Note that this is a breaking change, but we believe the type is never directly mentioned in any code
