    Class ApiCoreShim

    This is the base class for other base-classes of modern custom API Controllers.
    It provides Commands such as Ok() which already exist in .net core and allows hybrid controllers to have the same commands.

    Warning

    Do not inherit from this class, it's included in the docs so you can see the APIs the other base classes have.

    Use Custom.Hybrid.Api12 or Custom.Hybrid.Api14 as your real base classes
